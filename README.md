# 🌐 Primeiro Programa com HTML e JavaScript

## 📘 Sobre o Projeto
Projeto desenvolvido durante meus estudos iniciais de JavaScript integrado ao HTML.

O objetivo foi criar uma página simples com interação com o usuário utilizando caixas de diálogo.

## 🧠 Conceitos aplicados

- Estrutura básica do HTML
- Estilização com CSS
- Uso da tag <script>
- Interação com o usuário
- window.alert()
- window.confirm()
- window.prompt()

## 💻 Funcionalidade

A página exibe:

- Um título "Olá, Mundo!"
- Uma mensagem inicial
- Um alerta ao carregar a página
- Uma pergunta de confirmação
- Um campo para o usuário digitar seu nome

## ▶️ Exemplo de execução

Ao abrir a página:

1. Exibe um alerta: "minha primeira mensagem!"
2. Pergunta: "Está gostando de JavaScript?"
3. Solicita o nome do usuário
